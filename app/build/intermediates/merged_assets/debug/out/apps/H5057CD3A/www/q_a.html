Troubleshooting

  • 1.The battery fault light on

    If the battery fault light on, pls check the Troubleshooting number in Homepage, if the code is 0x1***, this problem would be recorved by itself. But if the code is 0x2*** or 0x3***, pls contact the AOBOET after service hot line or your distributor for help.
  • 2.The information of battery cannot be seen in the monitoring system?

    Check the battery status first, if the battery status is OFF, pls turn the battery on, then check the WLAN is accessiable for battery.
  • 3. Register the battery failure

    If Register the battery failure, pls check the network of mobile phone neraby the battery installation site available and stable.
  • 4. If there are other questions, pls refer the Installation Manual first or contact your distributor for help.
